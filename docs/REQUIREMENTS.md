# 旅行プラン作成アプリ 要件定義書

## 1. プロジェクト概要
Gemini 3.0 Flash の高速な生成能力を活かし、ユーザーの希望に合わせた旅行プランを瞬時に提案するWebアプリケーション。
「計画を立てるのが面倒」「現地の詳しい情報が知りたい」というユーザーをサポートし、素敵な旅の思い出作りをお手伝いするよ！

## 2. ターゲットユーザー
- 旅行は好きだけど、細かいスケジュールを組むのが苦手な人
- サクッと旅行プランを提案してほしい人
- おしゃれで使いやすいアプリで計画を管理したい人

## 3. デザイン・雰囲気
- **テーマカラー**: 白 (#FFFFFF) と 水色 (例: #E0F7FA, #00BCD4) を基調とした爽やかな配色
- **スタイル**: シンプル、モダン、余白を活かしたスマートなデザイン
- **デバイス対応**: スマートフォン完全対応 (レスポンシブデザイン)

## 4. 技術スタック
- **Frontend Framework**: Next.js 16.1 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS v4
- **AI Model**: Google Gemini 3.0 Flash (高速応答・マルチモーダル)
- **Deployment**: Vercel (予定)

## 5. 主な機能 (MVP: 最初のバージョン)

### 5.1 旅行プラン作成画面 (トップページ)
ユーザーが以下の情報を入力するフォーム。
- **行き先**: 国、都市、観光地など
- **日程**: 日数 (例: 2泊3日)
- **予算**: ざっくりとした予算感
- **誰と？**: 一人、友達、カップル、家族
- **スタイル**: のんびり、アクティブ、グルメ中心、映えスポット巡りなど
- **フリーテキスト**: その他の要望

### 5.2 プラン提案画面
AIが生成した旅行プランを表示。
- **タイムライン表示**: 1日ごとのスケジュールを時系列で見やすく表示
- **宿泊先提案**: おすすめの宿泊エリアやホテル名、特徴、目安の価格帯を表示
- **スポット情報**: 場所の名前、簡単な説明
- **アクション**: 「プランを再生成」「トップに戻る」

## 6. 今後の拡張アイデア (Ver.2以降)
- プランの保存機能 (データベース連携)
- Google Maps との連携で地図表示
- 持ち物リスト生成機能
- 旅のしおりPDF出力

## 7. 開発の進め方
1. **環境構築**: Next.js + TypeScript + Tailwind CSS
2. **UI作成**: トップページと結果画面の見た目を作る
3. **API連携**: Gemini API を組み込んで実際にプランを作れるようにする
4. **調整**: スマホで見やすく調整、AIのプロンプト改善
