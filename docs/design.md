# デザイン設計書 (design.md)

## 1. ビジュアルコンセプト
- **テーマ**: 「澄み渡る空と旅へのワクワク感」
- **キーワード**: 爽やか、スマート、直感的、清潔感
- **カラーパレット**:
  - Main: `#00BCD4` (シアンブルー) - 青空や海をイメージ
  - Accent: `#E0F7FA` (パステルブルー) - 背景やセクションの区切り
  - Base: `#FFFFFF` (ピュアホワイト) - 清潔感と読みやすさ
  - Text: `#37474F` (ダークグレー) - 柔らかい黒で視認性を確保

## 2. ユーザーインターフェース (UI)
- **全体構成**: 1カラムのモバイルファースト設計。
- **フォント**: 日本語が綺麗に見える Sans-serif (Noto Sans JP など)。
- **コンポーネント設計**:
  - **Header**: シンプルなロゴと、爽やかな水色のアンダーライン。
  - **Hero Section**: 「次はどこに行く？」というキャッチコピーと、直感的な入力フォーム。
  - **Form Elements**: 角丸（Rounded-2xl）で親しみやすく、水色のシャドウを薄く入れる。
  - **Plan Card**: タイムライン形式。左側に時間、右側にスポット情報のカードを配置。
  - **Hotel Card**: 宿の情報は特別な装飾（星アイコンなど）を付けて目立たせる。

## 3. ユーザー体験 (UX)
- **入力の簡易化**: 選択肢（チップ）を多く使い、スマホでもタップだけで入力できるようにする。
- **ローディングアニメーション**: 旅行かばんが揺れるような、可愛くて退屈させない待ち時間演出。
- **レスポンシブ**: PCで見ても横に広がりすぎないよう、最大幅を `max-w-2xl` 程度に制限して読みやすさを維持。
