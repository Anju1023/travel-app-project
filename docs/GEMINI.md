# プロジェクトのゴール

Gemini 3 を使って、旅行プランを作成するアプリを作りたい

# デザイン・雰囲気の希望

- 白と水色を基調とした爽やかなテーマ
- おしゃれでスマート
- スマホからも見やすい

# 使いたい技術

- Gemini 3.0 Flash
- Next.js 16.1
- TypeScript
- Tailwind CSS v4

# コーディングのルール

- コードの意味が分かるように、こまめに日本語コメントをつけること
- `any`型は使わず、ちゃんと型定義すること
- 複雑な書き方より、後で読み返してわかりやすい書き方を好みます
