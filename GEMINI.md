# Fuwari (ふわり) - AI Travel Planner Project (Completed! 🎉)

Gemini 3.0 Flash を活用した、おしゃれでスマートな旅行プラン提案アプリ。
「ふわり」と軽い気持ちで、新しい旅に出かけよう！あんじゅと一緒に作り上げました！

## 🔗 リンク

- **GitHub Repository**: https://github.com/Anju1023/travel-app-project
- **Live Demo (Vercel)**: https://vercel.com/anjus-projects-ee5add78/travel-app-project

## ✅ 達成した機能

- **AI プラン生成**: 最新の Gemini 3.0 Flash (preview) を使用。
- **詳細な条件設定**: 行き先、日程、予算、同行者に加え、「旅行時期」や「こだわり条件」も考慮。
- **Markdown 形式でのコピー**: 生成したプランを Notion や Slack にサッと共有できる「旅のメモ」機能を搭載！✨
- **UI/UX (Fuwari Theme)**:
  - **グラスモーフィズム**: すりガラスのような透明感と、空のような淡いグラデーションを採用した最新のデザイン。☁️
  - **究極のレスポンシブ**: スマホでの見やすさを徹底追求。変な改行や文字あふれを解消し、指先で心地よく操作できるステップドット UI を搭載。📱
  - **エモーショナルな演出**: 楽しいローディングアニメーション（弾むカバン 💼）や、ふわふわと浮遊するアイコンで旅のワクワク感を演出。
- **地図・持ち物連携**:
  - Leaflet を使用した地図表示で、目的地を視覚的に把握。📍
  - AI による「持ち物リスト」と「旅のアドバイス」で、準備も安心！🎒
- **堅牢な設計**:
  - Zod を使った AI レスポンスの厳格なバリデーション（防御力アップ！🛡️）
  - メンテナンスしやすいコンポーネント分割。
- **開発環境**: Next.js 16.1, TypeScript, Tailwind CSS v4

# プロジェクトのゴール

Gemini 3 を使って、旅行プランを作成するアプリを作りたい

# デザイン・雰囲気の希望

- 白と水色を基調とした爽やかなテーマ
- おしゃれでスマート
- スマホからも見やすい

# 使いたい技術

- Gemini 3.0 Flash
- Next.js 16.1
- TypeScript
- Tailwind CSS v4

# コーディングのルール

- コードの意味が分かるように、こまめに日本語コメントをつけること
- `any`型は使わず、ちゃんと型定義すること
- 複雑な書き方より、後で読み返してわかりやすい書き方を好みます
